    How to use ?
    Basic Features
  • To add new folders, files(text), album(for images) use right click on the screen to open menu.
  • To open a folder/file/album click on it once.
  • To check different features available with folder/file/album, right click on its icon.
  • In Text files, and albums hover over icons to get an idea of what that icon means.
  • Folders can be created in a nested structure. (Folder,files,albums inside other folders).
  • Home button in the bottom bar can be used to navigate to home.
  • List of all the folders/files/albums present can be accessed from by clicking their respective icon in bottom bar.
  • When a folder/file/album is clicked from list of all folders/files/albums view it will take you to its original location then it can be viewed from their.
  • Search bar can be used to search for folder/file/album names and contents of the text files.
  • Background can be changed (Access this option from right click menu).
    • Page Layout
  • Navigation bar on top the page represents the path to the folder.
  • Folder that is currently open has its name highlighted in the top bar.
  • Can go to a previous folder by clicking its name in top navigation bar.
  • Right click is disabled on navigation bar at the bottom.
